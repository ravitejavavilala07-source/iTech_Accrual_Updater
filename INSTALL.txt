uick Install & Usage

1) python -m venv .venv

2) source .venv/bin/activate   (on Mac/Linux)
   .venv\Scripts\activate       (on Windows)

3) pip install --upgrade pip

4) pip install -r requirements.txt

5) python app_polished.py

6) Mac ONLY - Make run_gui.command executable:
   chmod +x run_gui.command
   Then double-click run_gui.command to launch

7) pyinstaller --onedir --windowed --name iTechAccrual app_polished.py

- Mac: Excel from Microsoft Office
- Windows: Excel from Microsoft Office
